Everything

"Everything" is search engine (made by voidtools) that locates files and folders by filename instantly for Windows. Unlike Windows search "Everything" initially displays every file and folder on your computer (hence the name "Everything"). You type in a search filter to limit what files and folders are displayed.

SQL Server First Responder Kit

You're a DBA, sysadmin, or developer who manages Microsoft SQL Servers. It's your fault if they're down or slow. These tools help you understand what's going on in your server.

When you want an overall health check, run sp_Blitz. To learn which queries have been using the most resources, run sp_BlitzCache. To analyze which indexes are missing or slowing you down, run sp_BlitzIndex. To find out why the server is slow right now, run sp_BlitzFirst.
